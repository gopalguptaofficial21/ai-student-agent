# AI Student Query Assistant

A RAG-based AI agent that automatically answers student academic queries using FAISS vector database and Sentence Transformers.

## Features
- Answers student queries about exam dates, deadlines, attendance rules
- Uses RAG architecture for accurate information retrieval
- Vector similarity search using FAISS
- Sentence Transformer model for text embeddings
- Works with any custom knowledge base

## Technologies Used
- Python
- FAISS Vector Database
- Sentence Transformers
- RAG Architecture
- NumPy

## How to Run

Install dependencies:
pip install sentence-transformers faiss-cpu numpy

Create embeddings:
python embed.py

Run the agent:
python agent.py
